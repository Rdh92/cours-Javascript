Exo 13 - Parité

Soit la boucle for dans le js 13-parite.js : ce programme affiche les nombres pairs jusqu'a 10 dans la console. Tapez le et examinez le résultat dans votre console.

for (var i = 1; i <= 20; i++) {

if (i % 2 === 0) {

console.log(i + ' est pair');

}

}

Ce programme utilise le modulo (reste de la division est égal à 0) pour trouver les nombres pairs en dessous de 10

des exemples pour l'utilisation du modulo

  • console.log(10%2);// Affiche 0 car 10 = 5*2+0
  • console.log(11%2);// Affiche 1 car 11 = 5*2+
  • console.log(18%3);// Affiche 0 car 18 = 3*6+0
  • console.log(19%3);// Affiche 1 car 19 = 3*6+1
  • console.log(120%3);// Affiche 2 car 20 = 3*6+2
