TVA

Ecrivez un programme js qui fait saisir un prix HT (hors taxes) à l'utlisitateur, puis qui affiche le prix TTC (toutes taxes comprises) correspondant en se basant sur un taux de TVA à 20 %.

Vous pouvez effectuer l'affichage dans la console, dans une pop-up ou sur la page directement, dans un paragraphe avec un identifiant.
